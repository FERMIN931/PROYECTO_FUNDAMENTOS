README SIMPLE - EJECUTAR EL PROGRAMA

Programa:
Sistema Piñatería REYNA

Requisito:
Tener instalado Python 3 en la computadora.

Archivo del programa:
sistema_pinateria_rey.py

Pasos para ejecutar:

1. Guardar el archivo del programa en una carpeta.

2. Abrir la carpeta donde está el archivo.

3. En la barra de dirección de la carpeta, escribir:

cmd

y presionar Enter.

4. Ejecutar el programa con este comando:

python "sistema_pinateria_rey.py"

Si no funciona, probar:

python3 "sistema_pinateria_rey.py"

Uso del sistema:

Al ejecutar, aparecerá este menú:

PIÑATERÍA REYNA
1. Registrar producto
2. Listar productos
3. Registrar venta
4. Registrar pedido
5. Buscar producto
6. Generar reporte
7. Cargar 50 ventas
8. Salir

El usuario debe escribir el número de la opción que desea usar.

Importante:

Para cerrar correctamente el programa, usar la opción:

8. Salir

Así se guardan los datos en los archivos:

productos_rey.json
ventas_rey.json
pedidos_rey.json

Nota:

Si el nombre del archivo se cambia a:

sistema_pinateria_rey.py

el comando para ejecutar sería:

python sistema_pinateria_rey.py
